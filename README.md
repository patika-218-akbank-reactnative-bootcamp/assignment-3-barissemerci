# Telegram Clone
Telegram Clone projesi, bir kullanıcı oluşturularak giriş yapılan, mock bir kişiler listesinden listelenen kişilerin yer aldığı, tıklanılan kişiye mesaj göndermek, uygulamanın temasını değiştirmek ve giriş yapılan kullanıcının bilgilerini güncellemek ve çıkış yapmak gibi özelliklerin olduğu bir çalışmadır.
